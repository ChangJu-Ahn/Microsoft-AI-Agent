Microsoft Copilot Agent 실습
============
이 실습에서는 Microsoft 365 Copilot 에서 가장 쉽게 만들 수 있는 Agent Builder를 이용하여 나만의 Agent를 생성합니다. 
이는 Microsoft 365 제품 뿐 아니라, 3rd-party 그리고 우리 회사의 레거시 시스템을 대상으로 Agent를 생성할 수 있습니다.
