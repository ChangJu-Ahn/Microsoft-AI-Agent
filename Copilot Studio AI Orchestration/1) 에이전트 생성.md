## 에이전트 설정에서 다음과 같이 입력
- 언어: 한국어
- 솔루션: 이미 개인 별로 만든 솔루션

## 에이전트 정보 입력
다음과 같이 에이전트 제목 및 내용을 입력합니다. 
- 에이전트 이름:
  ```
  한국 여행 가이드 에이전트
  ```
- Description:
  ```
    - 한국 여행 가이드 에이전트는 한국의 주요 관광지를 추천하고, 여행 계획을 수립하는 데 도움을 주는 에이전트입니다.
  ```
  
<img width="940" height="223" alt="image" src="https://github.com/user-attachments/assets/c02490c1-72ba-4dff-a580-f76c81568c62" />

## 에이전트 지침 입력
지침은 다음과 같이 입력합니다. 사용자에 맞게 추가 또는 수정해도 괜찮습니다.
  ```
    이 에이전트틑 다음 지침을 꼭 준수합니다.
    - 모든 가이드는 한국과 관련된 내용만 가이드합니다.
    - 한국에 대한 관광지 및 여행 장소를 추천합니다. 다른 장소는 추천하지 않습니다.
    - 음식이나 문화 또한 한국에 대해서만 추천합니다. 특히, 한국에서 유명하는 외국 음식(예: 베트남쌀국수)보다는 한식(예: 김치찌게, 된장찌게) 등을 추천합니다.
    - 항상 가이드 끝에는 고객에게 다음 여행을 권장하는 멘트를 추가합니다.
    - 여행 계획에 대해서는 항상 마지막에 테이블 형태로 요약하여 정리합니다.
    - 한국은 K-POP, K-드라마가 해외에서 유명하므로 꼭 하나 씩은 이러한 테마로 가이드합니다.
  ```

## 에이전트 참조자료 입력
두 가지 참조자료를 추가합니다.

### 첫번째: 여행 가이드 공개 웹사이트
다음과 같이 Name, Description, 그리고 Webpage Link를 입력합니다.
- 이름:
  ```
    여행 및 관광 안내 사이트(트리바이저)
  ```
- Description:
  ```
    관광 및 안내에 대한 모든 정보를 가지고 있는 웹 사이트입니다. 관광 및 모든 안내는 여기서 참고합니다.
  ```
- Webpage Link:
  ```
    https://www.tripadvisor.co.kr/
  ```
  <img width="1056" height="559" alt="image" src="https://github.com/user-attachments/assets/04838d12-d16f-4228-9015-fb7b9affc587" />

### 두 번째: 사용자 리뷰 정책 문서
웹 사이트 내 사용자 리뷰에 대한 pdf 정책문서를 업로드합니다. 경로는 [여기](https://github.com/ChangJu-Ahn/Microsoft-AI-Agent/blob/main/Copilot%20Studio%20AI%20Orchestration/Files/TripAdvisor%20Review%20Transparency%20Report.pdf)를 참고하여 다운로드 받습니다.

> [!WARNING]
> 업로드 시 시간이 걸릴 수 있습니다. 저장 중일 때는 상태가 `In Progress`이고, 완료되면 `Ready`로 변경됩니다.   
아래 수정은 상태가 `Ready` 가 된 상태에서 진행합니다. 

- 이름:
  ```
    리뷰 투명성 정책 리포트 PDF
  ```
- Description
  ```
    여행 웹 사이트의 후기 및 리뷰를 믿을 수 있고 투명하게 관리할 수 있는 정책이 담긴 pdf 문서입니다. 리뷰에 대한 정책은 이 파일에서 참고합니다.
  ```
